## Sprite generation tools

The sprites in this character were created using an AI-assisted production pipeline. The following tools were used at various stages:

<AccordionGroup>
  <Accordion title="ChatGPT" icon="robot">
    Used for generating base character artwork and iterating on pose references during the initial sprite design phase.
  </Accordion>
  <Accordion title="Grok" icon="robot">
    Used for image generation tasks and exploring alternative visual styles for Raziel's spectral form.
  </Accordion>
  <Accordion title="Gemini" icon="robot">
    Contributed to the image generation workflow, particularly for generating base frames that fed into the 3D conversion step.
  </Accordion>
  <Accordion title="A2E" icon="wand-magic-sparkles">
    Used in the animation conversion stage to process and refine generated imagery into usable sprite frames.
  </Accordion>
  <Accordion title="Vidu" icon="film">
    Used for generating animation sequences as video output, which were then extracted frame-by-frame into BMP sprites.
  </Accordion>
</AccordionGroup>

### Sprite production pipeline

The full workflow from concept to in-game BMP frames:

<Steps>
  <Step title="Create base artwork">
    Generate a base character image using AI image tools (ChatGPT, Grok, Gemini).
  </Step>
  <Step title="Convert to 3D">
    Transform the 2D base into a 3D representation to enable consistent multi-angle animation.
  </Step>
  <Step title="Generate animations">
    Produce animated sequences for each move in video form using Vidu and A2E.
  </Step>
  <Step title="Convert video to frames">
    Extract individual frames from the animation video output.
  </Step>
  <Step title="Convert frames to BMP">
    Convert extracted frames to the BMP format required by MUGEN's SFF sprite system.
  </Step>
</Steps>

Raziel is a fan-made playable character for the **MUGEN** fighting game engine, based on the protagonist of *Legacy of Kain: Soul Reaver*. This version features Raziel in his **spectral (blue) form**, complete with the iconic Soul Reaver blade, multiple palette variants, and a full set of standing, crouching, and aerial attacks.

The project is community-driven, completely free, and open for modification and redistribution.

## Get started

<Steps>
  <Step title="Download the character files">
    Clone or download the repository from [GitHub](https://github.com/mugensworld1996/1.-Char_Raziel). The character folder is `Raziel_Char/`.
  </Step>
  <Step title="Copy into your MUGEN chars folder">
    Place the `Raziel_Char` directory inside your MUGEN installation's `chars/` folder so the path reads `chars/Raziel_Char/raziel.def`.
  </Step>
  <Step title="Add Raziel to your roster">
    Open `data/select.def` in your MUGEN installation and add `Raziel_Char/raziel` under the `[Characters]` section.
  </Step>
  <Step title="Launch MUGEN and play">
    Start MUGEN and select Raziel from the character select screen. Use palette 1 or 2 to switch between Reaver variants.
  </Step>
</Steps>

<Note>
  This character is in early development. Some animations and special moves are still being added. See the [Roadmap](/project/roadmap) for what's planned next.
</Note>
